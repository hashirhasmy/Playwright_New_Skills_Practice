## How to Use This Memory System

Memories help maintain context across sessions and track learnings from your R&D activities.

### Memory Types

**User**: Information about your role, goals, and expertise
**Feedback**: Patterns that work well or should be avoided
**Project**: Current initiatives, experiments, and findings
**Reference**: External resources and documentation links

### Adding Memories

As you discover new patterns, best practices, or solutions during your R&D work, memories will be created to track these findings for future reference.
